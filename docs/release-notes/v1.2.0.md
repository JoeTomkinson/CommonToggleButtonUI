# Release Notes - v1.2.0

**Release Date:** December 2024

---

## ?? New Features

### Custom Border Colors for Toast Notifications
Add a personalized touch to your notifications with customizable colored borders!

- **Enable/Disable Custom Border** - Toggle a colored border around toast notifications
- **Color Picker** - Choose any color using the Windows color picker dialog
- **Windows Accent Color Integration** - Defaults to your Windows 11 accent color for a seamless look
- **Reset to Accent** - One-click button to restore the Windows accent color
- **Adjustable Thickness** - Configure border thickness from 0.5 to 10 pixels

![Border Settings](../assets/border-settings-preview.png)

## ? Improvements

### Enhanced Settings UI
- **Wider Default Window** - Settings window now opens at 580px width (up from 520px) for better readability
- **Visual Separation** - Header and footer sections now have distinct background colors to clearly indicate non-scrollable areas
- **Larger Status Indicators** - The on/off status indicator circles are now larger and more visible:
  - Standard toast: 16x16px (previously 12x12px)
  - Compact toast: 14x14px (previously 12x12px)

### Fixed Auto-Update Mechanism
- **Corrected AppInstaller URLs** - Fixed the update check mechanism to properly detect new versions
- **Stable Update Endpoint** - Updates now check against a stable GitHub Pages URL rather than version-specific release URLs
- **Improved Documentation** - Added detailed comments explaining the correct update workflow

## ?? Technical Changes

### New Settings
Added `ToastBorderSettings` class with the following properties:
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `EnableCustomBorder` | bool | false | Enable/disable custom border |
| `CustomBorderColor` | string? | null | Hex color (e.g., "#0078D4") |
| `UseAccentColor` | bool | true | Use Windows accent color |
| `BorderThickness` | double | 2 | Border width in pixels |

### Theme Service Enhancements
- `GetWindowsAccentColor()` - Retrieves the current Windows accent color from system settings
- `ParseHexColor()` - Parses hex color strings to WPF Color objects
- `ColorToHex()` - Converts Color objects to hex strings for storage

## ?? Installation

### New Installation
Download the `.appinstaller` file for your architecture:
- **x64 (Intel/AMD 64-bit):** `ToggleNotifier.Package_x64.appinstaller`
- **x86 (32-bit):** `ToggleNotifier.Package_x86.appinstaller`
- **ARM64:** `ToggleNotifier.Package_ARM64.appinstaller`

### Upgrading from v1.1.x
If you have auto-updates enabled, the application will prompt you to update on next launch. Your settings will be preserved.
